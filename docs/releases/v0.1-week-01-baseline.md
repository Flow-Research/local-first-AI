# Release Notes: v0.1-week-01-baseline

## Purpose

`v0.1-week-01-baseline` marks the immutable Week 1 starter snapshot for new fellows joining the Local-First AI Context Assistant project.

## Use This Release When

- A new fellow joins after the main project has moved beyond Week 1.
- A fellow needs the original starter structure.
- The fellowship needs a stable reference point for onboarding.

## Start From This Release

```bash
git clone https://github.com/Flow-Research/local-first-AI.git
cd local-first-AI
git checkout v0.1-week-01-baseline
git checkout -b fellows/<name>/week-01
```

## Related Branch

The matching baseline branch is:

```text
baseline/week-01-starter
```

## Included In The Baseline

- 12-month roadmap structure.
- Weekly reports from Month 1 through Month 12.
- Research, design, features, benchmarks, docs, and source folders.
- Python, C, C++, and Rust source layout.
- Contribution guide.
- Pull request checklist.
- Verification script.
- Project framing image in the README.

## Rule

Do not use this release for ongoing shared work. Use it as a starter snapshot only.
