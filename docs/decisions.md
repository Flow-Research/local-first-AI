# Decisions

Use this file to record project decisions as they happen. Each decision should explain what was chosen and why.

## Decision 001: Use Python for Month 1

Reason: Python allows faster learning and prototyping.

## Decision 002: Use SQLite for local storage

Reason: SQLite is simple, local, file-based, and useful for local-first experiments.

## Decision 003: Start with laptop only

Reason: Adding Raspberry Pi, mobile, or embedded devices too early would increase complexity before the foundation is clear.
