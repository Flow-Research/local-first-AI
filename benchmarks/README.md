# Benchmarks

Benchmarks are organized by language and by output type.

| Folder | Purpose |
|---|---|
| `python/` | Python benchmark scripts |
| `c/` | C benchmark experiments |
| `cpp/` | C++ benchmark experiments |
| `rust/` | Rust benchmark experiments |
| `results/` | Raw benchmark outputs |
| `reports/` | Human-readable benchmark reports |

Benchmark results should be summarized in the matching weekly report.
