# Features

Use this folder to plan, track, and close feature work without making the codebase messy.

## Folder Meaning

| Folder | Meaning |
|---|---|
| `proposals/` | Feature ideas that are not active yet |
| `active/` | Current feature briefs being worked on in branches |
| `completed/` | Finished feature briefs after merge |

## Feature Brief Template

```markdown
# Feature: Name

## Branch

`feature/month-XX-week-YY-short-name`

## Problem

What problem does this feature solve?

## Scope

What is included?

## Out of Scope

What is intentionally not included?

## Research Input

Which paper, article, or source informed this work?

## Design Outcome

What design choice came from the research?

## Build Output

What code or artifact was created?

## Evidence

What proves it works?
```
