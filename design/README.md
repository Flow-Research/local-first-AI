# Design

Use this folder to capture design outcomes before and after implementation.

Design here means system design, interaction design, data design, and research-backed decisions.

## Folder Meaning

| Folder | Meaning |
|---|---|
| `outcomes/` | Written design outcomes and decision summaries |
| `sketches/` | Draft diagrams, flows, and rough design notes |

## Design Outcome Template

```markdown
# Design Outcome: Name

## Branch

## Problem

## Research Input

## Options Considered

## Chosen Design

## Why This Design

## Risks

## Evidence To Collect
```
