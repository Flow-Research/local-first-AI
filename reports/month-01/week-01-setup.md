# Month 1 Week 1: Setup and Project Definition

## 1. Goal for the Week

Define the local-first AI project, create the repository structure, and prepare the project for progressive weekly work.

## 2. Research / Learning

Paper or source: [Local-first software](https://www.inkandswitch.com/essay/local-first/)

Local-first computing means data should live on the user's device first. The system should remain useful without depending completely on the cloud.

I also learned that a good project structure makes the work easier to grow each week without becoming confusing.

## 3. Design Outcome

Design artifact: Initial repository structure and project architecture.

Decision: Organize the repo around research, design, features, source code, benchmarks, assets, reports, and durable docs.

Reason: The project is planned as a 12-month learning and build process, so the structure must support weekly progress without needing major restructuring.
